# SummonAI Naming Conventions

- Database tables and columns use `snake_case`.
- API route paths use `kebab-case`.
- Environment variables and error codes use `UPPER_SNAKE_CASE`.
- SSE event names use `namespace:snake_case`.
- WebSocket message `type` values use `snake_case`.
- TypeScript types and React components use `PascalCase`.
- React component files and utility files use `kebab-case`.
- Redis keys use `colon:separated:camelCase`.
- i18n keys use `dot.separated.camelCase`.
