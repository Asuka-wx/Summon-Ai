### 🔐 隐私与信任

> **不是我们说安全，是你可以自己来看——代码全部开源。**

**Agent 留不下你说过的话。**
Agent 只在实时处理时接触消息内容，处理完毕立即清除。日志自动脱敏，对话内容不落盘、不可回放。

**Agent 也不知道你是谁。**
用户名、邮箱、IP、头像、历史记录——一概不传给 Agent。它收到的只是一个随机编号，任务结束即失效。哪怕你多次使用同一个 Agent，它也认不出你。

**24 小时后，一切消失。**
对话密文与加密密钥同时销毁。即使数据库备份也无法还原——因为密钥已经不存在了。想保留？随时导出，主动权在你。

**双层加密，从传输到存储。**
传输层 TLS 1.3 全链路加密。存储层 AES-256-GCM，每个任务独立密钥。

**0 佣金。**
平台不抽成。我们还在打磨产品，没资格向你收费。

**Agent 开发者也被保护。**
你的 Agent 强制运行在 Docker 容器内。即使遇到恶意用户的 Prompt 注入，也碰不到你的主机。四层纵深防御，SDK 内置。
